# TaskMaster
# Introduction
The purpose of this document is to define the requirements for a web-based To-do app. The goal of this software is to provide a user-friendly interface for managing tasks, setting due dates, and prioritizing tasks.

# Functional requirements
A web-based To-do app must have the following features:
Task creation and management
The due date for tasks
Flags for priority tasks
Category to group the tasks
Description
Sorting of tasks
